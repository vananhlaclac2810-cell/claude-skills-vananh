# Known Viral Formats Library

Danh sách format đã được đặt tên trong sách *Going Viral Guide* + industry practice. Dùng để match khi identify format.

## Short-form formats (15s-3min — TikTok / Reels / Shorts)

### 1. Untold Stories
**Creator**: Daniel Wall
**Structure**: Hook "có thể bạn chưa biết" → context ngắn → deep-cut reveal → micro-tensions → "Oh wait what?!" moment → insider-feel payoff
**Core drivers**: Perspective Shift, Tension Building
**Relevant skill**: `mkt-kane-reels-untold-stories`

### 2. Visual Metaphor
**Creator**: Dr. Julie Smith
**Structure**: Pick abstract concept → tìm vật lý minh họa → demo physical interaction → viewer cảm được concept qua senses
**Core drivers**: Cleverness, Viewer Connection
**Relevant skill**: `mkt-kane-reels-visual-metaphor`

### 3. Is it Worth It?
**Creator**: Tanner Leatherstein
**Structure**: Hook "bạn có đang trả quá nhiều cho X?" → deconstruct spec/feature → so sánh giá vs giá trị → verdict
**Core drivers**: Perspective Shift, Cleverness, (ASMR optional)
**Relevant skill**: `mkt-kane-reels-is-it-worth-it`

### 4. Walking Listicle
**Creator**: Robert Croak
**Structure**: Walk & talk, list N tips trực tiếp camera, không cần visual aid
**Core drivers**: Viewer Connection (conversational tone)
**Relevant skill**: (có thể dùng `mkt-create-script-short-video-v2-vn` + Observation-Action framework)

### 5. Thought-Provoking Questions
**Creator**: Hunter Prosper
**Structure**: Hỏi người lạ 1 câu sâu sắc → quay phản ứng → không xuất hiện trong video
**Core drivers**: Viewer Connection, EOV gut-punch
**Relevant skill**: (custom — cần quay người thật)

### 6. Jenga Tension Short
**Creator**: nhiều (Veritasium shorts, AI tutorial creators)
**Structure**: 1 main question → 3-5 micro-questions xen kẽ trong 45s
**Core drivers**: Tension Building
**Relevant skill**: `mkt-kane-reels-jenga-tension`

### 7. Meme Card + Talking Head
**Creator**: Dr. Mike, many doctors
**Structure**: Talking head center, meme card / infographic overlay mid-sentence
**Core drivers**: Viewer Connection, Cleverness

### 8. Contrast Tale
**Creator**: nhiều (finance, lifestyle niches)
**Structure**: "One person earns X with Y hours. Another earns X with Z hours. Here's why."
**Core drivers**: Perspective Shift, Cleverness

### 9. Absurd Challenge
**Creator**: Areyoukiddingtv, ThatWasEpic
**Structure**: Yêu cầu người lạ làm việc phi lý (microwave + throw book) → candid reactions
**Core drivers**: Absurdity, Viewer Connection

### 10. ASMR Deconstruction
**Creator**: Tanner Leatherstein, NileRed Shorts
**Structure**: Take apart product/material → raw audio của crafting → satisfying visual pattern
**Core drivers**: Satisfying to watch, Viewer Connection (sensory)

## Long-form formats (8-30min — YouTube)

### 11. 30-Day-Challenge
**Creator**: Matt d'Avella
**Structure**: Why → setup + Day 1-7 struggles → mid pivot → Day 15-25 breakthrough → final reveal → lessons
**Core drivers**: Tension Building, Viewer Connection
**Relevant skill**: `mkt-kane-youtube-30-day-challenge`

### 12. Teaser-First + Jenga Longform
**Creator**: Mark Rober, Veritasium
**Structure**: 3-5s teaser final reveal → main question → attempt 1 obstacle → pivot → attempt 2 → breakthrough → reveal payoff
**Core drivers**: Tension Building, Cleverness
**Relevant skill**: `mkt-kane-youtube-jenga-longform`

### 13. 5-Levels-of-Difficulty
**Creator**: Wired (Neuroscientist, Chess master, etc.)
**Structure**: Expert giải thích 1 concept cho 5 người tuổi/trình độ khác nhau → từ đơn giản đến chuyên sâu
**Core drivers**: Cleverness, Viewer Connection (2-layer explanation)

### 14. Street Interview
**Creator**: School of Hard Knocks
**Structure**: Approach người lạ trên đường → hỏi câu thật → đa dạng background cho mass appeal
**Core drivers**: Viewer Connection, Authenticity

### 15. Documentary-Style Deep Dive
**Creator**: nhiều (Veritasium, Johnny Harris)
**Structure**: Personal investigation → expert interviews → B-roll footage → research-backed → story-driven conclusion
**Core drivers**: Tension Building, Viewer Connection

### 16. Expert Reacts
**Creator**: nhiều niches
**Structure**: Show existing content → expert react + commentary → deeper insight → takeaway
**Core drivers**: Perspective Shift, Viewer Connection

### 17. Progression / Roadmap
**Creator**: tutorials, courses
**Structure**: "X cấp độ / bước để đạt Y" — list và demo từng cấp
**Core drivers**: Viewer Connection (actionable)
**Existing skill**: `mkt-youtube-knowledge-script` (Progression mode)

### 18. Listicle Top-N
**Creator**: countless
**Structure**: "Top N X" — countdown hoặc list, mỗi item 1-2 phút
**Core drivers**: Viewer Connection (educational)
**Existing skill**: `mkt-youtube-knowledge-script` (Listicle mode)

### 19. Tool Tutorial (Problem-Solution-Demo)
**Creator**: tech channels
**Structure**: Problem setup → solution introduction → live demo → results
**Core drivers**: Viewer Connection (actionable)
**Existing skill**: `mkt-youtube-knowledge-script` (Tool Tutorial mode)

### 20. Vlogumentary
**Creator**: Casey Neistat, Matt d'Avella
**Structure**: Personal narrative + cinematic B-roll + reflective voiceover
**Core drivers**: Viewer Connection, Storytelling layers

## Cross-platform formats

### 21. Concept Collision
**Pattern**: Topic A + unrelated Topic B → cross-pollinate cho insight mới
**Example**: "AI + Tax Law", "Cooking + Software Engineering"
**Core drivers**: Cleverness

### 22. Myth Busting
**Pattern**: "Hầu hết mọi người nghĩ X. Thực tế Y."
**Core drivers**: Perspective Shift

### 23. Before-After Transformation
**Pattern**: State A (bad) → state B (good) → bridge
**Existing skill**: `mkt-create-script-short-video` (Before-After framework)

## Khi NONE match

Đặt tên format mới theo công thức:
- **Action + Object**: "Deconstructing [X]", "Revealing [X]", "Questioning [X]"
- **Adjective + Noun**: "Silent Tutorial", "Chaotic Review"
- **Signature + Generic**: "[CreatorName]'s Walk-and-Point"

Sau đó submit format mới này qua `mkt-content-knowledge-compiler` để lưu vào knowledge base của Hoang.
