# Pricing Psychology — Vietnamese Market

Phase 7 reference. Apply VND-specific pricing tactics.

## Quy tắc cốt lõi VN

1. **Charm pricing** kết thúc bằng `997` hoặc `99` hoặc `99K`
2. **Tránh số 4** ở cuối (kiêng kỵ chết)
3. **Số 8, 9, 6** mang ý nghĩa thịnh vượng → ưu tiên
4. **Tổng stack value ≥ 10× giá bán STANDARD**
5. **3-tier decoy** thay vì 1 giá duy nhất
6. **Payment plan** bắt buộc cho ticket > 3M VND
7. **Group buying** mentality → có referral discount

## 3-Tier Decoy Structure

Hormozi recommend 3 tiers. Vietnamese audience cũng phản ứng tốt với 3 lựa chọn (paradox of choice nếu nhiều hơn).

### Template chuẩn

```
┌─────────────────────┬─────────────────────┬─────────────────────┐
│ BASIC (Decoy thấp)  │ STANDARD (Sweet) ⭐ │ PREMIUM (Anchor)    │
├─────────────────────┼─────────────────────┼─────────────────────┤
│ Giá: [X]            │ Giá: [3-4× X]      │ Giá: [8-12× X]      │
│ Core only           │ Core + 3-4 bonus   │ Standard + 1-1 + VIP│
│ No support          │ Group support      │ Priority + private  │
│ Self-paced          │ Self + group       │ Hybrid + 1-1        │
│                     │ Lifetime           │ Lifetime + updates  │
└─────────────────────┴─────────────────────┴─────────────────────┘
```

**Goal split**: 10% BASIC, 70% STANDARD, 20% PREMIUM.

### Ví dụ thực tế

**Khóa học Marketing Online**:
- BASIC: 1.997.000đ — 12 module video, no community
- STANDARD: 4.997.000đ ⭐ — Module + Telegram group + 1 buổi Q&A/tuần
- PREMIUM: 12.997.000đ — Standard + 2 buổi 1-1 + audit landing page

**Coaching Business**:
- BASIC (Self-study): 7.997.000đ — Course + workbook
- STANDARD (Group): 24.997.000đ ⭐ — Course + weekly group call + community
- PREMIUM (1-1): 79.997.000đ — Standard + 12 buổi 1-1 + Voxer access

### Anti-pattern

❌ Decoy quá yếu (BASIC chỉ giảm 10% giá) → khách chọn BASIC nhiều
✅ Decoy phải "lossy" (BASIC thiếu phần khách thực sự muốn) → đẩy lên STANDARD
❌ PREMIUM chỉ đắt hơn không thêm value lớn → không ai chọn
✅ PREMIUM phải có cái BASIC + STANDARD không có (1-1, VIP access, done-with-you)

## Payment Plan Strategy

Bắt buộc cho ticket > 3M. Tăng conversion 20-40% ở thị trường VN.

### Format chuẩn

```
Trả một lần: 4.997.000đ
HOẶC
Trả 3 kỳ: 1.797.000đ/tháng (tổng 5.391.000đ — phí tiện ích 394K)
```

### Cách framing

- "Chỉ 1.797K/tháng — bằng 60K/ngày = 2 ly cà phê"
- "Chia 3 kỳ nhẹ nhàng, học ngay không chờ"
- "Lock giá hôm nay, trả dần 3 tháng tiếp theo"

### Quy tắc

- Phí tiện ích ~5-10% tổng giá (không quá 15%)
- Số kỳ thường: 2, 3, 6 (tránh 4)
- Mỗi kỳ phải end charm number (1.797K không phải 1.665K)
- Hỗ trợ MoMo/ZaloPay/banking auto-debit

### Khi nào KHÔNG dùng payment plan
- Low-ticket < 1M (overhead admin lớn hơn benefit)
- One-time mini-product (workshop 1 buổi)
- Audience corporate B2B (thường pay 1 lần)

## Group Buying & Referral

VN có văn hóa "rủ bạn cùng học". Tận dụng:

### Format

- **Referral discount**: "Rủ 1 bạn → cả 2 được giảm 10%"
- **Tribe pricing**: "Đăng ký từ 3 người: 4.497K/người thay vì 4.997K"
- **Founding member**: "30 founding member đầu: lifetime access không charge upgrade fee"

### Lưu ý
- Đừng giảm quá mạnh (>20% sẽ devalue offer)
- Có cap số lượng (chỉ 30 founding) → urgency thật
- Document rõ T&C để tránh dispute

## Bonus Pricing Psychology

### Stack display rules (cho landing page)

1. **List giá trị từng item rõ ràng** — không cộng dồn
2. **Cộng tổng giá trị bằng số to** — visual prominent
3. **Show giá bán bằng số đỏ/contrast cao**
4. **Highlight % saved** (không show số tuyệt đối)

### Sample format VN

```
✓ Core Program 12 Module            12.000.000đ
✓ BONUS 1: 50 Email Template         5.000.000đ
✓ BONUS 2: Quick-Win 7-Day Plan      3.000.000đ
✓ BONUS 3: 90 Ngày Telegram Group    7.000.000đ
✓ BONUS 4: 1-1 Onboarding Call       5.000.000đ
─────────────────────────────────────────────────
Tổng giá trị:                       32.000.000đ
Giá hôm nay:                         4.997.000đ
ANH/CHỊ TIẾT KIỆM:                  27.003.000đ (84%)
```

## Income Bracket Targeting

### Low entry (199K-997K)
- Students, mới ra trường, ngách hobby
- Phương thức: bank transfer, Momo, ZaloPay
- Trust low → cần guarantee mạnh + nhiều testimonial

### Mid market (1.997M-9.997M)
- Working professional 25-40, owner small biz
- Phương thức: bank transfer, credit card, payment plan 3 kỳ
- Trust moderate → testimonial + case study

### Premium (19.997M-99.997M)
- Business owner, agency, executive
- Phương thức: bank transfer (invoice + VAT), payment plan 3-6 kỳ
- Trust requires: video case study, founder authority, social proof platform

### Ultra-premium (100M+)
- Enterprise, C-level, high-net-worth
- Phương thức: contract + invoice + USD quoting
- Trust: warm intro, application, gated access

| **Anchor + reveal** | Mọi offer | "Giá trị 50M, hôm nay 4.997K" |

## Quy trình quyết giá (skill workflow)

1. Tính **stack value** sau Phase 4 (cộng core + tất cả bonus)
2. Chia 10 để có **floor price** (Hormozi minimum)
3. Round up to nearest charm number → **STANDARD price**
4. BASIC = STANDARD ÷ 3 (round charm), strip bonuses
5. PREMIUM = STANDARD × 2.5-3 (round charm), add 1-1/VIP
6. Verify không có số 4 ở cuối
7. Calculate payment plan: STANDARD ÷ 3 + 5% fee, round charm
8. Generate stack table với savings % displayed

**Sanity check trước khi finalize**:
- [ ] STANDARD giá có charm number không?
- [ ] Stack value / STANDARD ratio 8-12×?
- [ ] BASIC có thiếu yếu tố quan trọng (force upgrade)?
- [ ] PREMIUM có yếu tố exclusive (1-1, lifetime)?
- [ ] Payment plan available nếu giá > 3M?
- [ ] Không có số 4 ở final digit?
