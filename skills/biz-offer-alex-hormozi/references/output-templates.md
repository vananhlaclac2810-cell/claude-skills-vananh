# Output Templates — 3 Files Per Offer

Phase 7 reference. Skill viết 3 file vào `output/cases/<slug>/`:

1. `offer.md` — markdown report tiếng Việt full
2. `offer.json` — structured data cho downstream pipeline
3. `conversion-copy.md` — block headline + sub + CTA paste landing page

`<slug>` = kebab-case product name. VD `phuong-phap-cashflow-reset-90-ngay`.
