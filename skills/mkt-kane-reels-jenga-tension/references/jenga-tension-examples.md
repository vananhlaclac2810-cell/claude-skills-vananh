## Ví dụ 1 — "AI agent có thay được junior dev không?"

**Pillar**: P1 AI Demo & Tutorials
**Duration**: 60s
**Target**: SME manager / founder đang cân nhắc hire junior dev vs AI agent

### 3 MAIN Q Variations
1. "AI agent có thay được junior dev không?" (binary)
2. "Mình sa thải junior dev sau khi dùng Claude Code 30 ngày?" (shock)
3. "Junior dev hay AI agent — cái nào đáng đầu tư hơn?" (comparison)

**Recommended**: Q1 — rõ stakes, binary dễ hook

### Layered Plan
- **Layer 1 (debug)**: AI fix bug nhanh — nhưng bug gì?
- **Layer 2 (context)**: AI mất context giữa file — ai fix?
- **Layer 3 (maintenance)**: 6 tháng sau code AI viết, ai sửa?
- **Payoff**: AI thay được task, không thay được ownership

**[15-30s] LAYER 2 — context**

## Ví dụ 3 — "Fine-tune hay Prompting?"

**Pillar**: P1 AI Demo & Tutorials
**Duration**: 58s
**Target**: Dev / founder muốn custom AI cho business

### 3 MAIN Q Variations
1. "Fine-tune hay Prompting — cái nào đúng cho startup?" (comparison)
2. "90% startup chọn sai giữa Fine-tune và Prompting" (data shock)
3. "Tại sao fine-tune model $10K không bằng 1 prompt tốt?" (counter-intuition)

**Recommended**: Q2 — data shock + stakes cao

### Layered Plan
- **Layer 1 (cost)**: Fine-tune $5K-50K, Prompting $0 setup
- **Layer 2 (flexibility)**: Data đổi → fine-tune train lại; prompt đổi → 5 phút
- **Layer 3 (data need)**: Fine-tune cần 1000+ sample chất lượng; prompt cần 3-5 ví dụ
- **Payoff**: Chỉ fine-tune khi prompting fail — 90% case prompt đủ

### Full Script

**[0-3s] HOOK — [MAIN Q]**
"90% startup chọn sai giữa Fine-tune và Prompting. Mình đã sai 2 năm."
- Visual: talking head, caption "90% sai"
- Audio: beat drop

**[3-15s] LAYER 1 — cost**
[MICRO Q1] "Đầu tiên: cost. Fine-tune bao nhiêu?"
[ANSWER 1] "$5K đến $50K tùy model. Prompting $0 setup, chỉ trả token."
Transition: "Nhưng cost không phải vấn đề chính..."
- Visual: 2 cột số tiền, fine-tune cột cao đỏ, prompting cột thấp xanh
- Caption: "$5K-50K vs $0"

**[15-30s] LAYER 2 — flexibility**
[MICRO Q2] "Vậy khi business thay đổi — flexibility ra sao?"
[ANSWER 2] "Data update → fine-tune phải train lại, 1-7 ngày. Prompt update → sửa text, 5 phút."
Transition: "Và layer cuối ít ai nói..."
- Visual: icon clock — fine-tune 7 ngày vs prompt 5 phút
- Caption highlight: "Iteration speed"

**[30-45s] LAYER 3 — data need**
[MICRO Q3] "...data bạn có đủ để fine-tune không?"
[ANSWER 3] "Fine-tune cần 1000+ example clean. Startup thường có 50. Prompt cần 3-5 ví dụ few-shot — đủ rồi."
- Visual: dataset size diagram
- Audio: rising

**[45-55s] FINAL PAYOFF**
[FINAL PAYOFF] "Rule: chỉ fine-tune khi prompting fail sau 10 lần iterate. 90% case prompt + few-shot đủ. Fine-tune là cuối cùng, không phải đầu tiên."
- Visual: direct camera
- Audio: silence

**[55-60s] SOFT CTA**
"Mình có template prompt engineering trong AI Freedom Builders — link bio."
- Visual: soft smile
