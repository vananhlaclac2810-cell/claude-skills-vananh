## Pattern checklist (cho mọi script mới)

Khi review 1 script paste-tag listicle, check 9 điểm:

- [ ] Caption line 1 = `Comment '[KEYWORD]' ... 👇` (H2)
- [ ] Hook 0-3s có số + AI tool + outcome đo được (H3)
- [ ] Setup 3-6s nói rõ WHERE paste
- [ ] Đúng N tag (khớp với số ở hook)
- [ ] Tag name KHÔNG dịch (giữ English/technical)
- [ ] Benefit 1-line bắt đầu bằng động từ mạnh
- [ ] Loop tease ở 45-50s (lý do comment)
- [ ] "Save [bài này]" trong 10s cuối
- [ ] Last Dab không hype, không autocratic

## Anti-patterns (dấu hiệu đang lệch format)

❌ Tag name dịch sang tiếng Việt ("CHẾĐỘKHÔNGMƯỢT" thay vì "TRUTHMODE")
❌ Benefit dùng từ vague ("giúp năng suất hơn", "tốt hơn")
❌ Hook bỏ số ("Những Claude Code hook ẩn — paste vào settings.json")
❌ Thiếu lead magnet CTA ở caption line 1
❌ 5 tag mà body chỉ có 4 (hoặc ngược lại)
❌ Last Dab hype-driven ("Claude Code CỰC MẠNH!!!")
❌ Soft CTA autocratic ("Follow ngay!")
❌ Photo/link post thay vì reel (H5 violation — xuống Bronze)

## Lưu ý cross-language (Sabrina EN → Hoàng VN)

| Element | EN (Sabrina) | VN (Hoang) | Lý do |
|---------|--------------|------------|-------|
| Tag name | UPPERCASE tiếng Anh | **Giữ nguyên** UPPERCASE tiếng Anh hoặc tên technical | Tiếng Việt uppercase nhìn gắt; audience dev VN quen English tech terms |
| Nhân xưng | "I", "you" | "mình", "bạn" | Brand voice Hoang |
| Lead magnet | `Follow and comment 'CODES'` | `Comment 'HOOKS' để mình gửi` | Bớt autocratic, thêm lý do |
| Last Dab | Ownership flex | Ownership flex hoặc contrarian | Tone 7/10, không hype |
| Power words | System, Automation, Framework | **Giữ English**: System, Automation, Framework, Prompt, Template | Brand voice rule |
