## Example 1 — "AI Agent Automation" (Bucket + Drill)

**Duration**: 45s
**Pillar**: P1 AI Demo
**Prop**: Bucket đầy nước + máy khoan (drill) có mũi bơm
**Mapping**: Bucket nước = tasks lặp lại chồng chất / Drill = AI agent drain tasks

### Last Dab (viết FIRST)
"Bucket là việc của bạn. Drill là AI agent. Bạn chọn cái nào?"

### 3 Hook Variations
1. "Đây là AI agent automation giải thích bằng 1 cái bucket."
2. "Nếu bạn nghĩ AI agent phức tạp, xem cái này 45s."
3. "Bucket này là lý do bạn cần AI agent ngay."

**Recommended**: Hook 1 — clean, hint visual payoff.

### Script

**[0-3s] HOOK**
"Đây là AI agent automation giải thích bằng 1 cái bucket."
Camera: close-up bucket đầy nước
Audio: ambient

**[3-10s] PROP SETUP**
"Bucket này = việc lặp lại của bạn mỗi ngày. Reply email, lọc CV, update sheet, post FB."
Camera: wide, bucket ở giữa, sticky notes dán quanh ("email", "CV", "sheet", "FB")
Audio: ambient + nhẹ

**[10-35s] METAPHOR DEMO**
[10-15s Setup] "Mỗi ngày bucket đầy thêm."
Camera: đổ thêm nước vào bucket (close-up)
Audio: tiếng nước đổ raw

[15-30s Action] "Bạn có 2 lựa chọn. Múc bằng tay..."
Camera: múc bằng cốc nhỏ — chậm, bucket vẫn đầy
"...hoặc cắm drill này."
Camera: cắm drill có mũi bơm vào bucket
Audio: tiếng drill bật + tiếng nước hút mạnh

[30-35s Result] "30 giây. Bucket rỗng."
Camera: bucket rỗng, drill vẫn chạy
Audio: drill tắt — silence 1s

**[35-50s] INSIGHT**
"Đây là AI agent. Không phải chatbot. Nó có tay — drain bucket thay bạn. Trong lúc bạn ngủ, nó vẫn chạy."
Camera: face direct, bucket rỗng phía sau
Audio: subtle music in

**[50-60s] SOFT CTA**
"Bucket bạn đầy cỡ nào rồi? Comment xem mình reply. Nếu muốn build drill của riêng bạn, AI Freedom Builders có template — link bio."
Camera: face + bucket rỗng

## Example 2 — "Context Engineering" (Jar + Colored Sand)

**Duration**: 50s
**Pillar**: P1 AI Demo + P4 Mindset
**Prop**: Jar thủy tinh trong + 4 màu cát (trắng, đỏ, xanh, vàng) + 1 thìa
**Mapping**: Layer cát = layer context / Khuấy = mất context / Layer rõ = AI trả lời chính xác

### Last Dab
"Context không phải nhét nhiều. Là xếp đúng thứ tự."

### 3 Hook Variations
1. "Context engineering trông như thế này trong đời thực."
2. "Tại sao AI trả lời bạn sai? Nhìn jar này."
3. "Đây là sự khác biệt giữa prompt amateur và prompt pro — 50 giây."

**Recommended**: Hook 2 — trigger pain point trước khi demo.

### Script

**[0-3s] HOOK**
"Tại sao AI trả lời bạn sai? Nhìn jar này."
Camera: close-up jar rỗng
Audio: ambient

**[3-10s] PROP SETUP**
"Jar này = context bạn đưa vào AI. 4 màu cát này = 4 loại thông tin. Trắng = vai trò. Đỏ = task. Xanh = examples. Vàng = format output."
Camera: wide, 4 lọ cát xếp bên jar
Audio: ambient

**[10-35s] METAPHOR DEMO**
[10-15s Setup] "Prompt pro xếp từng layer rõ."
Camera: đổ cát trắng → đỏ → xanh → vàng tuần tự (close-up, slow)
Audio: tiếng cát rót raw

[15-25s Action 1] "AI nhìn jar này — thấy rõ layer nào, trả đúng layer đó."
Camera: jar có 4 layer màu rõ ràng
Audio: silence

[25-35s Action 2] "Prompt amateur? Ném tất cả vào 1 lần, thêm khuấy."
Camera: đổ hỗn hợp 4 màu 1 lần vào jar thứ 2 → lấy thìa khuấy mạnh
Audio: tiếng khuấy

[35s Result] "Bùn nâu. AI không biết đâu là đâu."
Camera: jar bùn nâu bên cạnh jar layered
Audio: silence 1s

**[35-50s] INSIGHT**
"Nhiều người nhét cả đoạn dài vào prompt rồi hỏi sao AI generic. Context engineering là xếp layer — không phải đổ nhiều. Jar 2 có cùng lượng cát jar 1. Khác ở thứ tự."
Camera: 2 jar cạnh nhau, face direct
Audio: subtle music

**[50-60s] SOFT CTA**
"Prompt gần nhất của bạn là jar 1 hay jar 2? Mình có template context xếp layer trong AI Freedom Builders — link bio."
Camera: face + 2 jar

## Example 3 — "Prompt Engineering" (Flashlight vs Laser Pointer)

**Duration**: 40s
**Pillar**: P1 AI Demo
**Prop**: Phòng tối + 1 đèn pin (flashlight thường) + 1 laser pointer + 1 target nhỏ (hình in trên tường cách 3m)
**Mapping**: Flashlight = prompt vague / Laser = prompt precise / Target = output mong muốn

### Last Dab
"Prompt không phải viết dài. Là bắn đúng."

### 3 Hook Variations
1. "Đây là lý do prompt của bạn ra output generic."
2. "Flashlight vs laser pointer — cái này giải thích prompt engineering."
3. "40 giây để hiểu tại sao prompt pro khác prompt amateur."

**Recommended**: Hook 1 — pain point + promise reveal.

### Script

**[0-3s] HOOK**
"Đây là lý do prompt của bạn ra output generic."
Camera: phòng tối, face hơi sáng
Audio: ambient

**[3-10s] PROP SETUP**
"Phòng tối = AI không biết gì. Đèn này = prompt của bạn. Target trên tường = cái bạn muốn AI làm."
Camera: wide, thấy target nhỏ trên tường
Audio: silence

**[10-35s] METAPHOR DEMO**
[10-20s Action 1] "Flashlight. Prompt kiểu 'viết cho tôi bài về AI'."
Camera: bật flashlight rọi tường — ánh sáng tản rộng, target chỉ được chiếu 1 phần
Audio: click đèn
"AI thấy cả tường — trả lời cả tường. Generic."

[20-30s Action 2] "Laser pointer. Prompt kiểu 'viết bài 300 từ về MCP server cho SME VN, giọng casual, ví dụ Notion MCP'."
Camera: tắt flashlight → bật laser → chấm đỏ đúng target
Audio: click laser, silence

[30-35s Result] "Chấm đỏ đúng target. AI trả đúng cái bạn muốn."
Camera: close-up chấm đỏ trên target
Audio: silence

**[35-50s] INSIGHT**
"Prompt pro không dài hơn — nó hẹp hơn. Flashlight dễ. Laser cần bạn biết target ở đâu. Biết target trước, viết prompt sau."
Camera: face direct, laser còn rọi target phía sau
Audio: subtle music

**[50-55s] SOFT CTA**
"Target prompt gần nhất của bạn ở đâu? Mình có 10 laser prompt template trong AI Freedom Builders — link bio."
Camera: face + target

## Cross-example patterns

1. **Prop có sound raw**: Mỗi example dùng âm thanh prop thật (nước đổ, cát rót, click đèn) — không music đè trong demo section.
2. **3 beats trong demo**: Setup → Action → Result. Mỗi beat 5-15s.
3. **Insight ≤25 từ**: Nói ít, để visual carry meaning.
4. **Last Dab tie back visual**: Reference prop cụ thể (bucket/jar/laser) — không generic advice.
5. **CTA có question + resource**: Hỏi user (engagement) + pointer đến AI Freedom Builders (non-autocratic).
