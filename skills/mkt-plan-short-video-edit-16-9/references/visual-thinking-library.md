# Visual thinking library — 25 metaphors

The soul of this skill. Every cream-paper editorial b-roll MUST pick a metaphor from this library (or `custom` with explicit warning). Each entry: when-to-use → layout sketch → on-image-text template → palette → rendered prompt example.

**Style baseline (applies to every entry):** warm cream `#F0EEE6` background, hand-drawn line illustration with subtle warm color fills, dark slate text, ALL Vietnamese with full diacritics (only brand names + tech symbols stay English), 16:10 aspect default. NOT parchment, NOT vintage, NOT photorealistic.

---

### dong-ho-cat (Đồng hồ cát đầy ↔ vơi)

**Best for:** BEFORE/AFTER scenes. Time savings. Productivity contrast. "Trước đây 8h → giờ 0h."

**Layout:** 2 đồng hồ cát side-by-side. Left = đầy cát màu rose, đứng yên, label "TRƯỚC ĐÂY · 8H". Right = vơi gần hết, label "HÔM NAY · 0H", màu lime. Mũi tên cong cream-edge nối giữa, label "Tự động" hand-written serif italic.

**Palette:** rose `#fb7185` (before), lime `#a3e635` (after), amber `#f4b860` (arrow accent).

**On-image text template:**
- Title: "TRƯỚC ĐÂY · {TIME_BEFORE} ↔ HÔM NAY · {TIME_AFTER}"
- Annotation 1: "Việc lặp đi lặp lại"
- Annotation 2: "Não mệt hơn tay"
- Annotation 3: "Sáng dậy việc xong"

**Anti-patterns:** Don't use digital countdown timer (sterile). Don't stack hourglasses vertically (compresses landscape framing). Don't add running clock animation indicator — this is still illustration, not motion.

### heo-dat-vo-ket-sat (Heo đất vỡ vs két sắt nhỏ gọn)

**Best for:** Cost/saving comparison. Result/recap scenes with money or efficiency framing. "−15h/tuần · +2 khách hàng."

**Layout:** LEFT = heo đất hồng nứt vỡ với xu vàng tumble ra ($, ₫). RIGHT = két sắt mini blue-grey gọn ghẽ, 1 xu trên nóc. Mũi tên cong rộng amber giữa, chữ "−15h/+2 khách" lớn.

**Palette:** rose `#f9a8d4` heo, slate `#4d6bfe` két, amber arrow, lime accent ✓.

**On-image text template:**
- Title: "{DELTA_LABEL} · {RESULT}"
- Subtitle: "Cũ nứt vỡ. Mới gọn ghẽ."
- Tags: "Tiết kiệm 15h/tuần", "+2 khách không thuê thêm ai", "Thời gian = tiền"

**Anti-patterns:** Don't use generic "money saved" pile-of-cash icon. Phải có vật thể đời thường ai cũng nhận ra.

### nga-ba-duong (Ngã ba đường)

**Best for:** Pivot scene. Decision moment. "Tôi đổi cách làm — không hỏi đáp nữa, mà giao việc."

**Layout:** Bản đồ hand-drawn cream. Đường trái dẫn vào màn hình chat "HỎI ĐÁP" rose, bị tắc cuối đường. Đường phải dẫn vào tower orb violet "GIAO VIỆC", open-ended. Pin ghim đỏ trên đường phải. Compass nhỏ góc.

**Palette:** rose `#fb7185` (path cũ), violet `#a78bfa` (path mới), amber pin.

**On-image text template:**
- Title: "RẼ HƯỚNG · TỪ HỎI ĐÁP SANG GIAO VIỆC"
- Biển báo cũ: "HỎI HOÀI"
- Biển báo mới: "GIAO LUÔN"

**Anti-patterns:** Don't use modern GPS app screenshot — must be hand-drawn paper map vibe.

**On-image text template:**
- Title: "CỬA MỞ · CONTEXT KHÔNG GIỚI HẠN"
- Tags: "Nhớ cuộc trò chuyện dài", "Không cần nhắc lại", "Phù hợp dự án nhiều tháng"

**Anti-patterns:** Don't use sci-fi portal (anh đã reject). Phải gỗ ấm.

### ong-nuoc-flow (Ống nước nối flow)

**Palette:** cyan, violet, claude-orange, lime accent.

**On-image text template:**
- Title: "TÀI LIỆU 100 TRANG → OUTPUT"
- Tags: "Không cần chunk", "Không cần re-prompt", "Một dòng chảy"

**Anti-patterns:** Don't use literal industrial plumbing. Style nhẹ nhàng cream paper, ống nước đơn giản như sketch trong notebook.

**On-image text template:**
- Title: "GHÉP TỪNG MIẾNG · MỘT KHỐI HOÀN CHỈNH"
- Tags: "Đầy đủ steps", "Không drop giữa chừng", "Multi-step OK"

### but-mau-ket-not (Bút màu kết nối — 2 bút bắt chéo)

**Best for:** Composition. Combining two things. "Prompt + memory = output."

**Layout:** Hand-drawn 2 cây bút bắt chéo trên giấy cream. Bút trái: chì grey, vẽ outline. Bút phải: bi đỏ, fill detail. Tạo thành sketch hoàn chỉnh ở giữa: chữ "GIAO VIỆC + BỐI CẢNH = OUTPUT".

**Palette:** slate (chì), rose (bi đỏ), warm cream paper.

**On-image text template:**
- Title: "NỐI Ý — BÚT CHÌ + BÚT BI"
- Subtitle: "Cùng nét, khác phong cách."
- Tags: "Prompt + memory", "Không phải là một, mà là cả hệ", "1 + 1 = 3"

**Anti-patterns:** Don't show 2 identical pens. Different types is the whole point.

**On-image text template:**
- Title: "ĐỨT MẠCH GIỮA CHỪNG"
